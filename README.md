Данное приложение предоставляет возможность вести заметки и помечать их степенью важности. В этой программе задействованы такие элементы разработки, как:

-RecyclerView
-Room
-RxJava
-viewHolder
-Adapter
-LiveData
-Паттерн singletone
-Паттерн MVVM
